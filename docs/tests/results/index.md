1. **在庫テストのタイムアウト**: 一部の在庫テストは完全実行時にタイムアウトしましたが、個別に実行した場合は成功しました。
2. **レポートテンプレートエラー**: レポートテストでThymeleafテンプレートエラーが観察されました。
3. **DeleteCategoryテスト**: UIに削除リンクが見つからないため、削除機能を適切にテストできませんでした。
