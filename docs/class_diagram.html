商品管理システム クラス図

本ドキュメントは、商品管理システムのクラス構造を説明するものです。

モデル (Model)
サービス (Service)
リポジトリ (Repository)
コントローラ (Controller)

※クラス図の詳細な内容はPlantUMLファイル（class_diagram.plantuml）を参照してください。

1. モデル (Model)

アプリケーションのデータ構造を表現するクラス

Product

商品を表現するモデルクラス

属性 型 説明
id Long 主キー
name String 商品名
janCode String JANコード
categoryId Long カテゴリID（外部キー）
price BigDecimal 価格
stockQuantity Integer 在庫数量
stockUnit String 在庫単位（個、箱など）
status String 商品状態（販売中、品切れなど）
description String 商品説明
createdAt LocalDateTime 作成日時
updatedAt LocalDateTime 更新日時
category Category 関連するカテゴリオブジェクト

Category

商品カテゴリを表現するモデルクラス

属性 型 説明
id Long 主キー
name String カテゴリ名
description String カテゴリ説明
createdAt LocalDateTime 作成日時
updatedAt LocalDateTime 更新日時

InventoryHistory

商品の入出庫履歴を表現するモデルクラス

属性 型 説明
id Long 主キー
productId Long 商品ID（外部キー）
quantityChange Integer 数量変化（正：入庫、負：出庫）
reason String 入出庫理由
operatedBy Long 操作者ID（外部キー）
createdAt LocalDateTime 作成日時
product Product 関連する商品オブジェクト
user User 関連するユーザーオブジェクト
メソッド 戻り値 説明
getType() String 入庫/出庫の区別を返す
getQuantity() Integer 変化量の絶対値を返す

User

システムユーザーを表現するモデルクラス

属性 型 説明
id Long 主キー
username String ユーザー名
password String パスワード（ハッシュ化）
fullName String 氏名
role String ロール（ADMIN, USER等）
enabled boolean 有効フラグ
createdAt LocalDateTime 作成日時
updatedAt LocalDateTime 更新日時

SystemSetting

システム設定を表現するモデルクラス

属性 型 説明
id Long 主キー
key String 設定キー
value String 設定値
description String 設定説明
createdAt LocalDateTime 作成日時
updatedAt LocalDateTime 更新日時

2. リポジトリ (Repository)

データの永続化を担当するインターフェース

ProductRepository

商品データの永続化を担当

メソッド 戻り値 説明
findById(id) Optional<Product> IDによる商品検索
findAll() List<Product> 全商品取得
findOutOfStock() List<Product> 在庫切れ商品取得
findLowStock() List<Product> 在庫少商品取得
save(product) Product 商品保存
deleteById(id) void 商品削除

CategoryRepository

カテゴリデータの永続化を担当

メソッド 戻り値 説明
findById(id) Optional<Category> IDによるカテゴリ検索
findAllSorted() List<Category> 全カテゴリをソートして取得
save(category) Category カテゴリ保存
deleteById(id) void カテゴリ削除

InventoryHistoryRepository

在庫履歴データの永続化を担当

メソッド 戻り値 説明
findByProductId(productId) List<InventoryHistory> 商品IDによる履歴検索
findLatestByProductId(productId, limit) List<InventoryHistory> 商品IDによる最新履歴検索
findByDateRange(startDate, endDate) List<InventoryHistory> 期間による履歴検索
findByType(type) List<InventoryHistory> タイプ（入庫/出庫）による履歴検索
save(history) InventoryHistory 履歴保存

UserRepository

ユーザーデータの永続化を担当

メソッド 戻り値 説明
findById(id) Optional<User> IDによるユーザー検索
findByUsername(username) Optional<User> ユーザー名によるユーザー検索
save(user) User ユーザー保存

SystemSettingRepository

システム設定データの永続化を担当

メソッド 戻り値 説明
findByKey(key) Optional<SystemSetting> キーによる設定検索
findAll() List<SystemSetting> 全設定取得
save(setting) SystemSetting 設定保存

メソッド 戻り値 説明

4. コントローラ (Controller)

ユーザーリクエストを処理し、適切なビューを返すクラス

ProductController

商品関連の画面制御

エンドポイント メソッド 説明
/products list(model) 商品一覧表示
/products/{id} detail(id, model) 商品詳細表示
/products/create createForm(model) 商品作成フォーム表示
/products/create create(product, redirectAttributes) 商品作成処理
/products/{id}/edit editForm(id, model) 商品編集フォーム表示
/products/{id}/edit update(id, product, redirectAttributes) 商品更新処理
/products/{id}/delete delete(id, redirectAttributes) 商品削除処理
